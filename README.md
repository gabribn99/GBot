# Comandos

```discord
--saluda
```
Hace que GBot salude

```bash
--perro
```
Envía una foto de un perro

```bash
--perro[comiendo,durmiendo,jugando]
```
Envía una foto de un perro realizando la acción indicada

```bash
--suma x y
```
Realiza la suma de x + y
